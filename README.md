## Getting Started

First, install package:

```bash
yarn install
# or
npm install
```

Then install ios package:

```bash
cd ios && pod install
```

Now run the development server :

```bash
yarn start
# or
npm run start
```

After that, running emulator Android or iOS:

```bash
yarn android
# or
yarn ios
```
